# 新討論串開場模板

每次開新串時，直接複製下面區塊，填空後貼上。

```text
專案：food-ai-app
本次目標：

先做這件事（單一任務）：

目前分支 / Commit：
- branch:
- latest commit:

已完成（最多 5 點）：
1.
2.

待辦（按優先序）：
1.
2.
3.

本次限制 / 不要動：
- 不要提交 docs/*、tools/*（除非我明確要求）
- 先不要做 UI 大改，先修功能

我要你先讀：
- docs/HANDOFF.md
- docs/SESSION_NOTES.md

驗證資訊（可選）：
- 平台：iOS / Web
- 重現步驟：
- 預期結果：
- 實際結果：
- 錯誤訊息（貼關鍵 20~40 行）：
```

## 使用建議（避免對話越來越慢）

1. 每個新串只做一個主目標。  
2. 長 log 只貼關鍵段落，不要整段貼。  
3. 截圖一次貼 1~2 張，必要再補。  
4. 功能做完就開新串，沿用這份模板。
